## Themeing

#### Variables
```css
:root {
  --ui-default-color: rgba(17, 24, 39, 1);
  --ui-default-bg: #ccc;
  --ui-primary-color: #fff;
  --ui-primary-bg: #34568B;
  --ui-secondary-color: #fff;
  --ui-secondary-bg: #222;
  --ui-disabled-opacity: 0.5;
}
```

#### css selectors

| name | description |
|---|---|
|`.ui.button`| For basic button |
|`.primary`| Primary button variant |
|`.secondary`| Secondary button variant |
|`.outline`| Outline button variant |
|`:disabled`| Disable state variant |
